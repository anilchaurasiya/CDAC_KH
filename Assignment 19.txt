1) Create a view that shows all of the customers who have the highest ratings.
Ans--> create view V1 as select * from customers where rating in(select max(rating) from customers);

2) Create a view that shows the number of salespeople in each city.
Ans--> create view v1 as select city, count(*) from salespeople group by city;

3) Create a view that shows the average and total orders for each salesperson after his or her name. Assume all names are unique.
Ans--> create view v1 as select sname, avg(amt) Average, count(amt) Total_Orders from orders, salespeople where 	       salespeople.snum=orders.snum group by salespeople.sname;

4) Create a view that shows each salesperson with multiple customers.
Ans--> create view v1 as select salespeople.* from orders,salespeople where salespeople.snum=orders.snum group by sname        having count(*) > 1;
